# ⛏️ マイニング疑似体験 Web

ブロックチェーン、特に **PoW（Proof of Work）** の仕組みを、座学ではなく **実際に手を動かして体感** するための学習用 Web サイトです。ハッシュ計算・ナンス探し・ブロックの連結・改ざん耐性を、ブラウザ上でさわりながら理解できます。

すべての計算はブラウザ内で完結します（サーバー・データベース・ユーザー登録は一切ありません）。

## 🎯 このサイトで学べること

1. **ハッシュ関数の性質** — 入力が1文字変わるだけで出力が全く変わる／出力から入力は逆算できない
2. **ナンス（nonce）探し** — 「先頭にゼロがN個並ぶハッシュ」を総当たりで探す作業こそがマイニング
3. **難易度（difficulty）** — ゼロの数が1つ増えるだけで計算量が跳ね上がる（約16倍）
4. **ブロックの連結** — 各ブロックが「前のブロックのハッシュ」を含むことで鎖になる
5. **改ざん耐性** — 過去のブロックを1文字でも書き換えると、それ以降の全ブロックが無効になる

## 📄 ページ構成

| パス | ページ | 内容 |
|---|---|---|
| `/` | トップ | サイト紹介と、おすすめ学習順（①→②→③）への導線 |
| `/hash` | ハッシュ実験室 | 入力をリアルタイムに SHA-256 化し、直前との差分をハイライト |
| `/mining` | マイニング体験 | **手動**（nonce を +1 して当たりを探す）と**自動**（Web Worker で総当たり＋統計）の2モード |
| `/blockchain` | ブロックチェーン | ブロックの追加・**改ざん実験**（連鎖無効化）・**再マイニング**での修復 |
| `/glossary` | 用語集 | ハッシュ・nonce・難易度・PoW などの初学者向け解説 |

> 💡 おすすめの学習順序は **① ハッシュ → ② マイニング → ③ ブロックチェーン** です。

## 🛠️ 技術スタック

- **ビルドツール**: [Vite](https://vitejs.dev/)
- **フレームワーク**: React 18 + TypeScript
- **スタイリング**: Tailwind CSS v4
- **ハッシュ計算**: Web Crypto API（SHA-256）— 追加ライブラリ不要
- **ルーティング**: React Router
- **状態管理**: React 標準（`useState` / `useEffect` / `useRef`）— 外部ライブラリなし
- **重い計算**: Web Worker でメインスレッドから分離し、マイニング中も UI が固まらない

完全な静的サイトなので、Vercel や GitHub Pages などの無料ホスティングにそのまま載せられます。

## 🚀 ローカルでの起動

Node.js（18 以上）が必要です。

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動（http://localhost:5173）
npm run dev

# 本番ビルド（型チェック + バンドル）
npm run build

# ビルド結果のプレビュー
npm run preview
```

## 📁 ディレクトリ構成

```
src/
├── main.tsx              # エントリーポイント（BrowserRouter でルーティングを有効化）
├── App.tsx               # ルート定義
├── index.css            # Tailwind の読み込みと全体テーマ
├── components/
│   ├── Layout.tsx        # 共通ヘッダー（ナビ）／フッター
│   └── HashDisplay.tsx   # ハッシュ表示（先頭ゼロを強調）
├── pages/
│   ├── HomePage.tsx
│   ├── HashPage.tsx      # ① ハッシュ実験室
│   ├── MiningPage.tsx    # ② マイニング体験（手動 / 自動）
│   ├── BlockchainPage.tsx# ③ ブロックチェーン（改ざん実験）
│   ├── GlossaryPage.tsx  # 用語集
│   └── NotFoundPage.tsx  # 404
├── lib/
│   ├── crypto.ts         # SHA-256 計算・難易度判定
│   ├── minerClient.ts    # マイニング用 Worker を扱うクラス
│   └── format.ts         # 数値・時間の表示整形
└── workers/
    └── miner.worker.ts   # nonce 総当たりのマイニング本体
```

コードには、初学者が読んで学べるよう **日本語コメント** を丁寧に入れています（特にハッシュ計算・マイニングループの部分）。
